<objective>
Implement the DDPG model-based critic (DDPG-02): a ResNet18-like CNN with a CoordConv stem and WeightNorm+TReLU backbone that maps the 7-channel rendered next-state `s_{t+1}` to a scalar value V(s'). This is the critical departure from standard DDPG — the critic takes the next-state image, NOT a concatenation of `(s_t, a_t)`.

Purpose: The critic estimates the value of the rendered next-state, which is what model-based DDPG (paper §3.3.1) uses for the Bellman target `y = r + γ·V_target(s_{t+1})`. WeightNorm+TReLU (not BatchNorm) per paper §3.4, and the modern `parametrizations.weight_norm` API is mandatory so the agent (Plan 03-04) can `copy.deepcopy(critic)` without crashing. Reuses the `CoordConv` and reuses the residual-stage structure from the actor (Plan 03-01).
Output: `models/critic.py` (TReLU, BasicBlockWN, Critic) + `tests/test_critic.py`, all tests green.

## Phase Goal

**As a** training-loop builder, **I want to** instantiate a critic that scores a rendered next-state with a single scalar V(s'), **so that** Phase 4 can compute the Bellman target and the deepcopy'd target critic stays deepcopy-safe.
</objective>

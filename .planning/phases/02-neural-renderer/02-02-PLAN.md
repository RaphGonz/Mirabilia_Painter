<objective>
Implement `pretrain_renderer.py` — the standalone supervised training script that (1) generates on-the-fly (params -> hard-rasterizer image) pairs with 20% extreme-param biasing, (2) trains `NeuralRenderer` by MSE against `draw(zeros, params)` to val MSE < 0.005, (3) saves `renderer.pkl` via `state_dict`, (4) runs the freeze assertion verifying R's param norm is unchanged after a frozen forward, and (5) exports `visual_gate.png` for human inspection. A blocking human-verify checkpoint confirms the visual gate before Phase 3 may begin.

Purpose: This is the Phase 2 HARD GATE deliverable. It produces the frozen renderer weights that Phase 4's `env.py` will load, and the visual + freeze evidence required by ROADMAP Success Criteria 2, 3, 4, and 5.

Output: `pretrain_renderer.py`, `renderer.pkl` (trained weights), `visual_gate.png` (comparison figure).
</objective>

<phase_goal>
**As a** developer building the DDPG painting agent, **I want to** have a pre-trained, visually validated, frozen neural renderer R, **so that** gradients can flow through stroke rendering during RL training without R drifting or producing unrecognizable strokes.

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| disk -> process (checkpoint load) | `torch.load('renderer.pkl')` deserializes a file. A tampered .pkl could execute arbitrary code via pickle. Treated as untrusted-deserialization boundary even though the file is self-produced. |
| R output -> downstream RL loss (Phase 4) | NaN/Inf in R output would propagate into the DDPG critic/actor loss. Boundary asserted finite at freeze-verification time. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-PKL | Tampering | `torch.load` of renderer.pkl | mitigate | Save only `state_dict` (tensors, no class/code). Load with `torch.load(path, weights_only=True)` in `load_frozen_renderer` and in all acceptance-criteria load commands. weights_only=True blocks arbitrary-code-execution via pickle (ASVS V7). |
| T-02-NaN | Tampering | NeuralRenderer.forward output feeding Phase 4 loss | mitigate | Sigmoid final layer bounds output to [0,1]; freeze verification runs `assert out.isfinite().all()` after load, before the checkpoint is accepted. |
| T-02-FREEZE | Tampering | R parameters drifting during downstream RL | mitigate | `load_frozen_renderer` calls `.eval()` + `requires_grad_(False)`; `param_norm` assertion (< 1e-6 delta after forward) proves no gradient flowed (REND-03 / Success Criterion 4). |
| T-02-SC | Tampering | npm/pip/cargo installs | accept | Zero new packages installed (RESEARCH.md Package Legitimacy Audit: torch/torchvision/matplotlib/tqdm/numpy all approved in Phase 1). No install task present; package-legitimacy checkpoint does not apply. |
</threat_model>

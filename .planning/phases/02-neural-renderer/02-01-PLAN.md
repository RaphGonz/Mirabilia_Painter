<objective>
Implement the differentiable neural renderer R as `models/renderer.py::NeuralRenderer` — an FC + bilinear-upsample CNN decoder mapping stroke params `(batch, 8)` to stroke images `(batch, 3, 64, 64)` in `[0, 1]` with no BatchNorm (per D-09/D-10/D-11). Create the Wave 0 test scaffold `tests/test_neural_renderer.py` first (RED), then implement the class to make the shape/range/no-batchnorm/single-sample tests pass (GREEN).

Purpose: R is the differentiable proxy that lets gradients flow through stroke rendering during DDPG training (Phase 4). This plan delivers a complete, importable, shape-correct vertical slice — after it, a user can instantiate R, run a forward pass, and get correctly-shaped bounded output. Pretraining (weights) comes in Plan 02-02.

Output: `models/renderer.py` (NeuralRenderer class), `tests/test_neural_renderer.py` (all REND-01 tests plus the REND-03 freeze-assertion test, which exercises freeze logic without file I/O).
</objective>
